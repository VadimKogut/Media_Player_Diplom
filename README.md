# Media_Player_Diplom

**Z-Player** — это современный мультимедийный плеер, разработанный на C++ и Qt. Поддерживает видео и аудио файлы, имеет интуитивный интерфейс, горячие клавиши и возможность кастомизации темы оформления.

Дипломный проект студента Московского Авиационного Института, группа М3О-433Б-22.

## ✨ Возможности

### 🎬 Воспроизведение
- Поддержка популярных форматов: `MP4`, `MKV`, `AVI`, `MOV`, `MP3`, `WAV`, `OGG`, `FLAC`
- Регулировка скорости воспроизведения: **0.25x — 4x**
- Перемотка на **±10 секунд**
- Поддержка субтитров
- Режим **Loop** (повтор текущего файла)
- Режим **Shuffle** (случайный порядок)

### 🎮 Управление
- **Горячие клавиши** (работают независимо от раскладки клавиатуры):
  - `Space` — Play / Pause
  - `F` — Полный экран
  - `←` / `→` — Перемотка ±10 сек
  - `↑` / `↓` — Громкость ±5%
  - `N` / `P` — Следующий / Предыдущий трек
  - `M` — Выключить звук
  - `Esc` — Выход из полноэкранного режима

### 🎨 Интерфейс и кастомизация
- Автоматическое скрытие панелей управления при просмотре
- Анимация кнопок при наведении и нажатии
- **Смена иконок плеера** (4 готовых стиля: Regular dark, Pretty dark, Soft Glow, Modern white)
- Полная настройка темы оформления:
  - Цвет фона, кнопок, рамок, слайдеров
  - Ширина и высота кнопок
  - Стиль и радиус рамки
- Готовые пресеты тем: Light, Dark, Ocean, Sand

### 📁 Управление файлами
- Автоматическое создание плейлиста из всех файлов той же папки
- Открытие файла через командную строку

## 📦 Сборка и запуск

### Требования
- Qt 5.15+ или Qt 6.x (с модулями: `core`, `gui`, `widgets`, `multimedia`, `multimediawidgets`)
- Компилятор с поддержкой C++17 (MinGW, MSVC, Clang)

### Сборка через qmake

```bash
# Клонируем репозиторий
git clone https://github.com/VadimKogut/Media_Player_Diplom.git
cd Media_Player_Diplom

# Собираем проект
qmake Media_Player_Diplom.pro
make

# Запускаем
./Z-Player
